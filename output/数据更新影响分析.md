## 关键发现

### 1. 分品牌ADR更新影响有限
- 美居ADR下调约10%(550→495.58)，但OCC略升
- 全季ADR上调约11%(400→443.91)，OCC也略升
- **综合影响: 广州客房收入仅+149万元/年(+1.2%)**

### 2. 房产税影响最大
- **广州: 房产税从540万增至1,257万(+717万/年)**
- **上海: 房产税从180万增至348万(+168万/年)**
- 合计增加约885万元/年税费支出

⚠️ **注意**: noi_engine.py第280行的房产税计算有单位转换问题，当前代码:
```python
hotel_prop_tax = hotel_prop_base * hotel_prop_rate / 10000
```
这会将结果错误地缩小10000倍，需要修正为:
```python
hotel_prop_tax = hotel_prop_base * hotel_prop_rate
```

### 3. 土地使用税变化较小
- 广州: -2.8万元/年
- 上海: -2.04万元/年
- 相对房产税影响可忽略

### 4. 总体影响
| 项目 | NOI变化 |
|------|---------|
| 广州 | 减少约566万元/年 |
| 上海 | 减少约165万元/年 |
| 合计 | 减少约731万元/年 |

## 对DCF估值的影响

以5.75%折现率、19.28年(广州)/30.65年(上海)计算:

- **广州项目PV减少**: 约566万 × 12.5年(年金因子) ≈ **7,075万元**
- **上海项目PV减少**: 约165万 × 15.8年(年金因子) ≈ **2,607万元**
- **合计估值减少**: 约 **9,682万元**

相比原估值(约15.91亿元)，**估值下调约6.1%**。
