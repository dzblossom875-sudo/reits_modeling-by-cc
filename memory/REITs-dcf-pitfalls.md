# 酒店类REIT DCF建模常见陷阱与核对清单

## 陷阱1: GOP数据来源错误

| 项目 | 说明 |
|------|------|
| **问题** | 利润表GOP vs 管理报表GOP口径不同 |
| **核对** | 明确提取的是哪个报表的数据 |
| **修正** | 管理报表更贴近实际经营，优先使用 |

**如何区分：**
- 管理报表GOP：通常包含更完整的经营数据
- 利润表GOP：会计口径，可能有调整项

---

## 陷阱2: 评估基准日混淆

| 项目 | 说明 |
|------|------|
| **问题** | 混淆2025年数据 vs 2026年数据 |
| **核对** | 明确评估基准日（如2025-12-31） |
| **修正** | 基准日之后的第一年是DCF首年（如2026年） |

**关键检查点：**
```
评估基准日: 2025-12-31
DCF首年: 2026年 (t=1)
土地使用权剩余年限: 从2025-12-31起算
```

---

## 陷阱3: 增长率假设单一化

| 项目 | 说明 |
|------|------|
| **问题** | 不同项目使用相同增长率 |
| **核对** | 检查招募说明书是否分项目披露增长率 |
| **修正** | 按项目分别设置增长率 |

**示例（不同项目不同增长率）：**
```python
growth_rates = {
    "project_a": {"y2": 0.02, "y3": 0.03, "y4_10": 0.025, "y11+": 0.02},
    "project_b": {"y2": 0.01, "y3": 0.02, "y4_10": 0.025, "y11+": 0.02}
}
```

---

## 陷阱4: 折旧重复扣除

| 项目 | 说明 |
|------|------|
| **问题** | GOP已扣除折旧，NOI计算又扣一次 |
| **核对** | GOP是否包含折旧？（通常包含） |
| **修正** | GOP - 折旧 = 经营现金流，不要再扣折旧 |

**公式确认：**
```
GOP = 营业收入 - 营业成本 - 税金及附加 - 管理费用 - 折旧
NOI = GOP + 商业净收入 - 其他费用 - Capex  (不再扣折旧)
```

---

## 陷阱5: 对比基准错误

| 项目 | 说明 |
|------|------|
| **问题** | 与募集资金对比（包含资本结构） |
| **核对** | 明确对比目标是资产评估值 |
| **修正** | 始终与资产评估值对比 |

**区分：**
- 资产评估值：纯资产价值（DCF应该接近这个）
- 募集资金：包含发行费用、资本结构安排等

---

## 陷阱6: 终值处理分歧

| 项目 | 说明 |
|------|------|
| **问题** | 使用永续增长模型 vs 持有到期模型 |
| **核对** | 土地使用权到期后是否有残值？ |
| **建议** | 保守起见使用持有到期（残值=0） |

**推荐做法：**
- 酒店REIT通常采用**持有到期模型**
- 土地使用权到期后残值 = 0
- 不进行永续增长假设

## 建模前核对清单

### 数据提取阶段
- [ ] 确认招募说明书版本（草稿/终稿）
- [ ] 标记所有参数来源页码
- [ ] 区分管理报表 vs 利润表数据
- [ ] 确认评估基准日

### 参数确认阶段
- [ ] 各项目土地使用权剩余年限
- [ ] 各项目分段增长率
- [ ] 折现率/报酬率
- [ ] 首年及预测期Capex

### 建模阶段
- [ ] GOP数据来源标注清楚
- [ ] 折旧是否已剔除确认
- [ ] 房产税/土地税处理确认
- [ ] 分段增长率实现正确

### 验证阶段
- [ ] 与资产评估值对比
- [ ] 单房估值合理性检查
- [ ] 隐含Cap Rate合理性
- [ ] 敏感性分析覆盖关键参数

---

## 模型审计报告必备内容

1. **Input层级图**: 明确哪些是直接摘录，哪些是计算得出
2. **参数来源表**: 每个参数的来源页码
3. **假设分类表**: 哪些是招募书给定，哪些是用户假设
4. **计算步骤详解**: 公式、示例、逐年计算
5. **对比分析**: DCF估值 vs 资产评估值
6. **敏感性分析**: 折现率和增长率敏感性
7. **风险提示**: 关键假设风险

---

## 快速问题诊断

| 症状 | 可能原因 | 检查点 |
|------|----------|--------|
| DCF估值远高于评估值 | 折旧重复扣除 / Capex未扣除 | 检查NOI计算公式 |
| DCF估值远低于评估值 | 增长率设置过低 / 折现率过高 | 核对增长率参数 |
| 单房估值异常 | 房间数错误 / 总估值计算错误 | 检查房间数单位 |
| 隐含Cap Rate异常 | NOI口径错误 | 确认NOI计算逻辑 |
