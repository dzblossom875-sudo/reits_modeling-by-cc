# 酒店类REIT DCF建模工作流程

## 标准工作流

```
PDF读取 → 参数标记 → 参数确认 → 建模 → 模型审计 → 估值比对 → 差异记录
```

### 各阶段具体动作

| 阶段 | 动作 | 输出 |
|------|------|------|
| **PDF读取** | 使用提取工具获取文本和表格 | 原始提取文件 |
| **参数标记** | 标记页码、区分Input/Assumption/Calculation | 标记后的参数表 |
| **参数确认** | 与用户确认关键假设（增长率、折现率） | 确认记录 |
| **建模** | 编写DCF模型代码 | Python模型脚本 |
| **模型审计** | 生成审计报告，明确三类数据 | 审计报告 |
| **估值比对** | 与招募说明书评估值对比 | 对比表 |
| **差异记录** | 记录差异原因和修正历史 | 差异分析文档 |

## 酒店类REIT核心参数清单

### Tier 1: 项目基础信息（必须提取）

- [ ] 项目名称、房间数、建筑面积
- [ ] ADR（日均房价）- 历史3年数据
- [ ] OCC（入住率）- 历史3年数据
- [ ] RevPAR（若有）
- [ ] **土地使用权剩余年限**（关键！影响DCF年限）
- [ ] 开业时间

### Tier 2: 财务数据（DCF直接Input）

- [ ] GOP（经营毛利）- **区分管理报表口径 vs 利润表口径**
- [ ] 营业收入（客房、餐饮、租赁等明细）
- [ ] 运营支出（现金部分）
- [ ] 折旧（非现金项，需从GOP中识别）
- [ ] 房产税、土地使用税（是否从NOI扣除需确认）
- [ ] 资本性支出（Capex）- 至少前3年预测值
- [ ] 商业净收入、其他费用（调整项）
- [ ] **NOI/CF（现金流口径）- DCF起点**

### Tier 3: 估值参数（必须核对）

- [ ] 折现率/报酬率（招募说明书通常披露）
- [ ] 分段增长率（**注意不同项目可能有差异！**）
- [ ] Cap Rate（倒算参考）
- [ ] 资产评估值（对比基准）

## DCF建模关键步骤

### Step 1: 数据准备
- 从`extracted_params.json`读取数据
- **关键**: 区分不同项目的剩余年限（如广州19.28年 vs 上海30.65年）
- **关键**: 确认GOP数据来源（管理报表 vs 利润表）

### Step 2: 推导首年NOI（REITs口径）
```
营业收入 = first_year_amount（已是不含增值税净额）
运营成本 = Σ运营明细项 + 物业费(独立合同) + 保险(独立合同)
GOP = 营业收入 - 运营成本 - 税金及附加(实际缴纳值)
管理费 = GOP × fee_rate（通常3%，付给酒店管理公司）
NOI = GOP - 管理费
NOI/CF = NOI - Capex
```
- **禁止**: 增值税重复扣除（营业收入已不含增值税）
- **禁止**: 成本口径混用（REITs明细 vs 历史利润表只能选一种）
- **禁止**: 用利润表"管理费用"替代 GOP×fee_rate
- 注意：折旧已从GOP中剔除，不要再扣除
- 注意：税金及附加使用实际缴纳值，推导值仅对照参考
- 注意：推导NOI始终使用，不回退到招募值（差异>5%标注但仍采用）

### Step 3: 增长率设置
- **必须核对**: 不同项目可能有不同的增长率
- 分段增长率常见结构：第2年、第3年、第4-10年、第11年起

### Step 4: 现金流生成
```
基础NOI = 推导NOI（来自Step 2）
每年NOI = 基础NOI × 累积增长因子
每年FCF = 每年NOI - 当年Capex
```

### Step 5: 折现计算
```
PV = Σ(FCF_t / (1 + r)^t)   t=1..N，含部分年
```
- 持有到期模型：残值 = 0（土地到期无残值）
- 不使用Gordon永续增长模型
- 剩余年限不取整，含部分年按比例计算

### Step 6: 汇总与对比
- 总估值 = 各项目估值之和
- 单房估值 = 总估值 / 总房间数
- 隐含Cap Rate = 首年NOI / 总估值
- **对比基准**: 资产评估值（不是募集资金）

## 输出文件清单

标准输出应包括：
- `dcf_summary.csv` - 估值汇总
- `dcf_cashflows.csv` - 现金流明细
- `dcf_sensitivity.csv` - 敏感性分析
- `dcf_results.json` - 完整结果
- `DCF模型审计报告.md` - 审计文档
- `估值差异分析.md` - 差异说明
