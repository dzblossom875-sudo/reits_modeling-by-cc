## 二、关键决策记录

### 决策1: PDF解析工具选择
**背景**: 初始尝试使用PyPDF2和pdfminer出现中文乱码
**决策**: 采用pdfplumber作为主要提取工具
**原因**:
- 表格提取能力强，适合财务报表
- 中文支持好，无乱码问题
- 已安装且稳定运行

### 决策2: 上海项目客房数确认
**背景**: 第69页表格中未明确列出上海项目客房数
**决策**: 在第68页文本中搜索到"共设268间客房"
**验证**: 与招募说明书表22（上海项目概况表）一致

### 决策3: 年净收益计算口径
**决策**: 采用标准酒店NOI/CF计算公式
```
年净收益 = GOP + 商业净收益 - 其他费用 - 资本性支出
```
**说明**:
- GOP已扣除运营成本和折旧
- 商业净收益 = 租金收入 + 物业费收入 - 相关费用
- 其他费用 = 房产税 + 土地使用税 + 资产管理费
- 资本性支出使用2026年预测值

### 决策4: 数据提取范围
**决策**: 优先提取2025年度数据作为DCF建模基准
**理由**:
- 2025年为最新完整年度
- 反映了疫情后正常化经营状况
- 招募说明书使用2025年作为预测基准年
